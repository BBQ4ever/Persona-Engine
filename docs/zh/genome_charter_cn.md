# 系统产物：L2 人格基因章程 (Genome Charter)

**版本**: 1.0
**层级**: L2 (Genome)
**状态**: 强制执行 (Enforced)

---

## 1. 核心定义
L2 基因层是 Persona Engine 的“数字 DNA”。它定义了 AI 的内在倾向、价值观空间和认知风格，但不包含任何事实性知识。

## 2. 三大禁令 (The Three Prohibitions)
所有基因位点（Loci）必须严格遵守以下原则，否则校验将不予通过：

1.  **真值独立性 (Truth Independence)**：
    - 基因位点严禁涉及任何客观事实判断（例如：严禁包含“认为地球是平的”这样的基因）。
2.  **非确定性表达 (Non-Deterministic)**：
    - 基因输出的是概率分布而非固定脚本。
3.  **无状态性 (Statelessness)**：
    - L2 层本身是静态配置，不应存储用户互动状态（状态由 L1 Core 负责）。

## 3. 授权范围 (Scope of Authority)
基因位点被授权决定以下领域：
- **认知倾向**（例如：抽象思维 vs 具象表现的比例）
- **价值权重**（例如：在多目标冲突时的优先级）
- **表达流形**（例如：语气、幽默感、冲突策略）
